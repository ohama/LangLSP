# Requirements: FunLang LSP

**Defined:** 2025-02-03
**Core Value:** LSP 입문자가 실제 동작하는 Language Server를 만들면서 LSP 개념을 이해할 수 있는 실용적인 튜토리얼

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### LSP Foundation

- [x] **LSP-01**: VS Code에서 FunLang 파일(.fun) 열면 LSP 서버 자동 시작
- [x] **LSP-02**: 문서 열기/닫기/변경 이벤트 동기화 (Document Sync)
- [x] **LSP-03**: LSP 서버와 VS Code 간 JSON-RPC 통신

### Diagnostics

- [x] **DIAG-01**: 문법 오류 실시간 표시 (빨간 밑줄 + 메시지)
- [x] **DIAG-02**: 타입 오류 실시간 표시
- [x] **DIAG-03**: 파일 저장 없이 타이핑 중 오류 업데이트

### Hover

- [x] **HOVER-01**: 변수 위에 마우스 올리면 추론된 타입 표시
- [x] **HOVER-02**: 함수 위에 마우스 올리면 함수 시그니처 표시
- [x] **HOVER-03**: 키워드 위에 마우스 올리면 설명 표시

### Completion

- [x] **COMP-01**: 키워드 자동 완성 (let, if, match, fun 등)
- [x] **COMP-02**: 현재 스코프의 변수/함수 자동 완성
- [x] **COMP-03**: 타입 정보와 함께 완성 항목 표시

### Go to Definition

- [x] **GOTO-01**: 변수 클릭 시 정의 위치로 이동
- [x] **GOTO-02**: 함수 호출 클릭 시 함수 정의로 이동
- [x] **GOTO-03**: 같은 파일 내 정의로 이동

### Find References

- [x] **REF-01**: 변수의 모든 사용 위치 찾기
- [x] **REF-02**: 함수의 모든 호출 위치 찾기
- [x] **REF-03**: 결과를 References 패널에 표시

### Rename Symbol

- [x] **RENAME-01**: 변수 이름 일괄 변경
- [x] **RENAME-02**: 함수 이름 일괄 변경
- [x] **RENAME-03**: 변경 전 미리보기 제공

### Code Actions

- [x] **ACTION-01**: 사용하지 않는 변수 제거 제안
- [x] **ACTION-02**: 타입 오류에 대한 수정 제안

### VS Code Extension

- [x] **EXT-01**: FunLang 문법 강조 (TextMate grammar)
- [x] **EXT-02**: 언어 설정 (주석, 괄호 자동 닫기)
- [x] **EXT-03**: 코드 스니펫 (let, match, fun 등)
- [x] **EXT-04**: .vsix 파일로 패키징

### Testing (Expecto + FsCheck)

- [x] **TEST-01**: 테스트 프레임워크 설정 (Expecto 10.2.3+, Expecto.FsCheck 10.2.3+)
- [x] **TEST-02**: Document Sync 단위 테스트
- [x] **TEST-03**: Diagnostics 단위 테스트 (문법 오류, 타입 오류)
- [x] **TEST-04**: Hover 단위 테스트
- [x] **TEST-05**: Completion 단위 테스트
- [x] **TEST-06**: Go to Definition 단위 테스트
- [x] **TEST-07**: Find References 단위 테스트
- [x] **TEST-08**: Rename Symbol 단위 테스트
- [x] **TEST-09**: Code Actions 단위 테스트
- [x] **TEST-10**: LSP 통합 테스트 (mock client)
- [x] **TEST-11**: FsCheck 속성 기반 테스트 (위치 변환, 범위 계산 등)

### Tutorial

- [x] **TUT-01**: LSP 기초 개념 설명 (프로토콜, 아키텍처)
- [x] **TUT-02**: F# LSP 라이브러리 선택 (Ionide vs OmniSharp 비교)
- [x] **TUT-03**: F# 프로젝트 설정 튜토리얼
- [x] **TUT-04**: Document Sync 구현 튜토리얼
- [x] **TUT-05**: Diagnostics 구현 튜토리얼
- [x] **TUT-06**: Hover 구현 튜토리얼
- [x] **TUT-07**: Completion 구현 튜토리얼
- [x] **TUT-08**: Go to Definition 구현 튜토리얼
- [x] **TUT-09**: Find References 구현 튜토리얼
- [x] **TUT-10**: Rename Symbol 구현 튜토리얼
- [x] **TUT-11**: Code Actions 구현 튜토리얼
- [x] **TUT-12**: VS Code Extension 패키징 튜토리얼
